不改变原数组的方法：

1.indexOf() 和lastIndexOf()

1.1 indexOf() : 返回元素在数组的索引，从0开始。若数组不存在该元素，则返回-1。

	var arr = [1, 2];
	arr.indexOf(1); //0
	arr.indexOf(10); //-1

1.2 lastIndexOf(): 返回元素在数组中最后一次出现的索引，如果没有出现则返回-1.
